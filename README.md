# Ag Equipment Serial Number Decoder

This project aims to reverse engineer and decode Agricultural Equipment serial numbers, mapping them to their corresponding Year, Make, and Model.

## Project Structure
- `data/`: Place raw datasets, CSVs, or text exports here.
- `docs/`: Reference material, notes from the decoding book, and logic rules.
- `src/`: Python scripts, parsing logic, and decoding algorithms.
- `notebooks/`: Optional Jupyter notebooks for interactive analysis and pattern discovery.
